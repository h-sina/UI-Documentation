sina-ui的说明文档

使用说明：

  • npm install sina-ui || npm i sina-ui
  • import 'sina-ui/dist/sina-ui.css';
  • import sinaui from 'sina-ui';
  • app.use(sinaui);

组件：

button

使用： <n-button></n-button>
API 说明
:msg='data.msg' 设置按钮文字
@click='clickEvent' 设置按钮点击事件
:shape='round|circle|默认' 设置按钮形状
:color='red' 设置按钮字体颜色
:disabled='true|false' 设置按钮禁用状态

carousel

使用： <n-carousel> <n-carousel-item></n-carousel-item> </n-carousel>
API 说明
:width='width（500px）' 设置宽度
:height='height（200px）' 设置高度
:speed='speed（1/2/3）' 设置走马灯转换速度
